# MovieApp

A modern movie web app built with React, Tailwind CSS, and TMDB API. Users can search for movies, explore popular movies (under the All Movies section), filter by categories, and view detailed movie information when users click on the card. The app also tracks trending searches using Appwrite database and a fully responsive UI for desktop and mobile devices.

Features:

* Browse popular and trending movies
* Trending section based on real user searches stored in Appwrite
* Filter movies by categories like Action, Comedy, Horror, Sci-Fi, and Romance
* Interactive movie details modal with overview, rating, language, and release year
* Responsive design optimized for desktop and mobile devices
* Smooth UI with hover effects and horizontal trending scroll
* Dynamic movie posters and fallback posters for missing images
* Fast search experience using debounced API requests
